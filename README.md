# Just_Eat
An algorithmic groceries shopper that automatically plans your meals based on nutritional guidelines and uses DoorDash API or Kroger API to order groceries automatically; a cheaper alternative to HelloFresh or similar

## Documentation & Software Design

All documentation for the project is located in the `Docs` directory. This folder contains the initial project concept and the formal requirements.

*   **[Initial_Idea.md](Docs/Initial_Idea.md)**: A high-level overview of the project, including the app's purpose, unique features, and technical considerations.
*   **[Specifications_Analysis](Docs/Specifications_Analysis/)**: You can find documents breaking down the functional and non-functional requirements derived from the initial idea. 

### Documentation Structure

For quick navigation, you can refer to the following file structure:

```text
Docs/
└── Initial_Idea.md                                  # High-level project concept
└── Specifications_Analysis/    
    ├── Specification for Requirements Analysis.md   # Detailed requirements document
    └── Specs_sheet_example.md                       # Example template for specsheets
```
